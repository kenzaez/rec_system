# Beauty Recommendation System

A Django-based e-commerce platform with AI-powered product recommendations tailored to user preferences.

## Features

- **Personalized Recommendations** – ML-driven product suggestions based on user history and ratings
- **Smart Search** – Filter by category, rating, and price with multiple sort options
- **User Authentication** – Secure login/registration with demo account access
- **Shopping Cart** – Add/remove products with checkout and order management
- **Wishlist** – Save favorite products for later
- **Product Ratings** – Rate and review products with recommendation impact
- **Viewing History** – Track recently viewed products
- **Chatbot Support** – Built-in customer assistance

## Tech Stack

- **Backend:** Django
- **Database:** Django ORM
- **ML Service:** Custom recommendation engine
- **Frontend:** HTML/CSS/JavaScript

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
5. Visit `http://localhost:8000`
